# VanJet Sound Assets

This directory contains audio files for the admin panel.

## Required Files

### new-order.mp3
A notification sound played when a new order is received.

**Recommended specifications:**
- Format: MP3
- Duration: 1-3 seconds
- Size: < 100KB
- Style: Pleasant, attention-grabbing chime

**Free sources for notification sounds:**
- https://notificationsounds.com/
- https://mixkit.co/free-sound-effects/notification/
- https://freesound.org/

Place the audio file as `new-order.mp3` in this directory.
